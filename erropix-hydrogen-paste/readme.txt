This plugin allows you to paste any elements you have previously copied using Hydrogen Pack

== Description ==

Hydrogen Paste allows you to paste any elements copied using [Hydrogen Pack](https://www.cleanplugins.com/products/hydrogen-pack/ "The premium version") across all your websites.

You can also paste any Oxygen elements shared by other users of the Oxygen community.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/erropix-hydrogen-paste` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Open the Oxygen Editor and start pasting elements
